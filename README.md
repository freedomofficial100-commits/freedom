# Freedom Official — بواريك شعر طبيعي

موقع تحويل من تيك توك لواتساب لمتجر **Freedom Official** لبواريك الشعر الطبيعي.

## 🚀 النشر على Vercel

### الطريقة ١ — ربط GitHub بـ Vercel (الأسهل)

1. ارفعي هذا المجلد على GitHub
2. ادخلي [vercel.com](https://vercel.com) وسجلي دخول
3. اضغطي **"Add New Project"**
4. اختاري الـ repository من GitHub
5. Vercel يكتشف الإعدادات تلقائياً من `vercel.json`
6. اضغطي **Deploy** — خلاص!

### الطريقة ٢ — Vercel CLI

```bash
npm i -g vercel
cd freedom-official
vercel
```
